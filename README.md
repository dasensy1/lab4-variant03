# Salary Calculator - Вариант 3



## Описание

Пакет `salarycalc` предназначен для расчета зарплаты сотрудников. Он позволяет вычислять:

- Общую зарплату до вычета налогов (Gross Salary)
- Чистую зарплату после налога (Net Salary)
- Применять бонусы
- Формировать отчет по зарплате

Проект использует внешний пакет [github.com/fatih/color](https://github.com/fatih/color) для цветного вывода отчета в консоли.
